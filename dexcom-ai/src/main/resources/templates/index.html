This project showcases
working proof of AI
using G7 data to benefit users

Dexcom Developer API

Fetch user data via public API endpoints and store it in MongoDB using a Spring Boot Kotlin app.
Note: This experiment uses sandbox data.

Train model

Using a simple locally hosted Prophet model in a container. Initial data is used to train model and additional data can be used to train model.

Automated dev-ops

Domain name to deployment in cloud is all done via terraform with infrastructure code. Docker and kubernetes is used.

Skills demonstrated

Skill Description

SpringBoot Swagger, Resilience4j, Hazelcast, Actuator & spring profile is used
